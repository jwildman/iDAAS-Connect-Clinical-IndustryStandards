Downloading Kafka

Download the latest source code from Red hat: https://access.redhat.com/products/red-hat-amq?extIdCarryOver=true&sc_cid=701f2000001OH7EAAW

Starting Kafka

1. Unzip the file into the directory (for our example and all the scripts we will show it is /Users/developer/RedHatTech/kafka_2.12-2.4.0.redhat-00005) 2. Start ZooKeeper bin/zookeeper-server-start.sh config/zookeeper.properties & 3. Open a New Terminal window 4. Change directory to /Users/developer/RedHatTech/kafka_2.12-2.4.0.redhat-00005 5. Start up Kafka bin/kafka-server-start.sh config/server.properties & Thats it - Kafka is running and ready now, there is no UI or Web UI. However, you could download a few different releases through Git Hub or other sources.
